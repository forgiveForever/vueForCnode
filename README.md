## react-client
- 前置
   1. 阅读doc目录文档
- 项目需要用到的技术
     - webpack
     - vue
     - vuex
     - scss
     - element-ui
     - babel
     - localStorage/sessionStorage

- 使用  
  - 安装依赖 `npm install & yarn`

- 运行
   - 编译 `npm run build`
   - 启动 `npm start`

- debug.json
   - servers : 后台服务器地址
   - dev : 本地开发环境

- 分析
    - 运行 `webpack --profile --json > stats.json`
    - http://alexkuz.github.io/webpack-chart/
    - http://webpack.github.io/analyse/
